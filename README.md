# react-template
with webpack, webpack-dev-server, babel-loader

## 使用方式 ##

1.  下載 react-template
    
    > git clone https://github.com/FayeHuang/react-template.git
    
    > cd react-template

2.  安裝相關套件

    > npm install

3.  啟動 webpack-dev-server

    > PORT=8080 npm start

  如果要介接 mock server

    > MOCK_SERVER=http://10.10.10.10:8888 PORT=8080 npm start
